# README — Prompts Padrões da Biblioteca IA Dev

## Objetivo

Prompts prontos para acionar a IA seguindo:

```txt
Prompt → orquestrador.md → controle-projeto.md → docs/ → protocols/ → speciality/ → skills/ → execução → documentação/checkpoint
```

## Regra principal

Todo prompt relevante deve exigir:

- ler `orquestrador.md`;
- ler `controle-projeto.md`;
- ler `docs/00-controle-consistencia-projeto.md`;
- usar o protocolo correto;
- convocar especialistas corretos;
- aplicar skills corretas;
- criar/alterar arquivos necessários;
- atualizar documentação em `docs/`;
- atualizar `controle-projeto.md`.
