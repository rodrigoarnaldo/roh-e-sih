# README — Docs do Projeto

## Objetivo

A pasta `docs/` guarda a documentação oficial do projeto.

Ela deve ser usada junto com:

```txt
orquestrador.md
controle-projeto.md
prompts/
protocols/
speciality/
skills/
```

## Diferença

```txt
controle-projeto.md = checkpoint, etapa atual e próximo passo
docs/ = verdade oficial do projeto
```

## Ordem de leitura recomendada

```txt
1. controle-projeto.md
2. docs/README.md
3. docs/00-controle-consistencia-projeto.md
4. docs/01-visao-geral-projeto.md
5. documento específico da tarefa
```
