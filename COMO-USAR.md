# Como Usar

## 1. Para começar projeto novo

Use:

```txt
prompts/02-iniciar-projeto-zero.md
```

Depois:

```txt
prompts/19-criar-estrutura-inicial-arquivos.md
```

## 2. Para continuar projeto existente

Use:

```txt
prompts/03-continuar-de-onde-parou.md
```

## 3. Para converter projeto externo

Use:

```txt
prompts/04-converter-projeto-externo.md
```

## 4. Para deploy

Use:

```txt
prompts/21-planejar-release-versionamento.md
prompts/15-homologacao.md
prompts/22-auditoria-seguranca-pre-producao.md
prompts/16-deploy-producao-easypanel.md
```

## 5. Regra final

Ao final de qualquer tarefa relevante, use ou exija:

```txt
prompts/20-atualizar-controle-e-docs.md
```
